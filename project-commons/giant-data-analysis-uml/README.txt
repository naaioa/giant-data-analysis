# 1 如何使用

Use UML Designer Plugin, REF: https://marketplace.eclipse.org/content/uml-designer

reserve `.project` file.
// change dotproject to .project.

In eclipse:
(1) import as regualr project;
(2) change project facets using 'Migrate to UML Designer 4.0'
(3) observe in `Modeling` perspective and `Model Explorer` view.
